# business-coach-ui-ux-design
This is a *sample UI/UX concept project* designed using Figma. The project focuses on creating a clean, modern, and user-friendly landing page for a business coach website.

## Project Type
- Sample / Practice Project
- Non-client work

## Tools Used
- Figma

## Objective
To practice layout structure, visual hierarchy, and CTA placement
for a business-oriented landing page.
